  2. .enumerate()

    Sat 01 August 2015
    By Kieran Hunt

    Rust, like Python, has a great feature called enumerate.

    Enumerate allows for you to access the current value of an iterator as well as its index.

    let foo = vec![1, 35, 64, 36, 26];
for (i, item) in foo.iter().enumerate() {
    println!("The {}th item is {}", i+1, item);
}
